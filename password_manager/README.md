# Password Manager CLI

A simple command-line password manager built with Python.

## Features

- Add new passwords
- View saved passwords
- Search passwords by website
- Delete passwords
- JSON file storage

## Project Structure

```
password_manager/
│
├── main.py
├── manager.py
├── storage.py
└── passwords.json
```

## How to Run

```
python main.py
```

## Example

```
1. Add Password
2. View Passwords
3. Search Password
4. Delete Password
5. Exit
```

## Technologies Used

- Python
- JSON
- CLI
